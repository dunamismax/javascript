About This Application

This Todo List application is a full-featured task management system built with modern web technologies. It demonstrates the power of the JavaScript monorepo architecture with shared utilities and FlareJS router integration.

Key Features

  • Create, read, update, and delete tasks
  • Priority levels (High, Medium, Low)
  • Category organization (Work, Personal, Shopping, Health, General)
  • Advanced filtering and search capabilities
  • Real-time statistics and analytics
  • Fast SPA-like navigation with FlareJS router
  • Persistent SQLite database storage
  • Responsive design with dark theme

Technology Stack

  • Backend: Node.js, Express.js, SQLite3
  • Frontend: Vanilla HTML, CSS, and JavaScript (ES Modules)
  • Router: FlareJS for enhanced navigation
  • Styling: Custom CSS with design system
  • Architecture: npm workspaces monorepo
  • Shared Utilities: @dunamismax/utils package

FlareJS Integration

This application showcases FlareJS router capabilities:

  • Lightning-fast page transitions
  • Intelligent link prefetching
  • Preserved JavaScript state between pages
  • Enhanced user experience with SPA-like navigation
  • Minimal overhead (only 2kB gzipped)

API Endpoints

GET /api/todos Retrieve all todos with optional filtering
POST /api/todos Create a new todo item
PUT /api/todos/:id Update an existing todo
DELETE /api/todos/:id Delete a todo item
GET /api/stats Get analytics and statistics

Monorepo Structure

This app is part of a larger JavaScript monorepo that includes:

  • Shared utility packages (@dunamismax/utils)
  • UI component library (@dunamismax/ui-components)
  • FlareJS router (@dunamismax/flare-router)
  • Shared configuration (@dunamismax/config)
  • Weather dashboard application
