About This Application

This Weather Dashboard provides real-time weather information for cities around the world. Built with modern web technologies and enhanced with FlareJS router for lightning-fast navigation.

Features

  • Real-time weather data from OpenWeatherMap API
  • Clean and responsive user interface
  • Fast SPA-like navigation with FlareJS
  • Temperature, humidity, wind speed, and visibility data
  • Weather icons and descriptions

Technology Stack

  • Node.js & Express.js backend
  • Vanilla HTML, CSS, and JavaScript
  • FlareJS router for enhanced navigation
  • Vanilla JavaScript (ES Modules)
  • CSS with custom design system

Data Source

Weather data is provided by OpenWeatherMap API, offering accurate and up-to-date weather information from around the globe.
